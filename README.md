# Metody Monte Carlo

Między innymi:

- *Podstawy i generowanie rozkładów:* skalowanie błędu statystycznego, własne generatory liczb losowych (1D i 2D), metoda akceptacji-odrzucenia, transformacja Boxa-Mullera, algorytm Metropolisa, weryfikacja testami $\chi^2$.
- *Całkowanie, wariancja:* całkowanie mocno oscylujących funkcji, techniki redukcji wariancji, losowanie warstwowe (stratified sampling), alokacja optymalna Neymana.
- *Błądzenie losowe i dyfuzja:* symulacje ruchów Browna w zamkniętych geometriach, tensor dyfuzji.
- *DSMC:* modelowanie oddziałującego gazu (kinetyczna teoria gazów), zderzenia sprężyste twardych kul, termalizacja od ścianek (rozkład Maxwella-Boltzmanna), odtwarzanie parametrów makroskopowych (gęstość, pole prędkości).
